A system to manage electroencephalogram

In this project it is asked to create an information system capable of receiving and visualising EEG files from distinct proveniences, no matter the manufacturers. The system should provide a multi-tenant platform, DICOM and EDF compatible with tools to manage, visualise and report EEGs in the web enviroment.
